# Paycheck Splitter

A tiny, single-file web tool that helps you split a paycheck (or any income deposit) into categories like:
- Spending / gas
- Core investing (e.g., VOO / VTI)
- Dip fund (cash reserved for market drawdowns)
- Crypto (optional)

It runs entirely in the browser — no backend, no accounts, no tracking.

## Live Demo
If you enable GitHub Pages, your site will be available at:

`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

## Files

- `index.html`  
  The main version (you can personalize defaults for your own use).

- `index-template.html`  
  A blank, annotated template meant for other people to fork and customize.

## How to Use

1. Open the site.
2. Enter your payment amount (e.g., your paycheck).
3. (Optional) Enter a monthly extra deposit (if you have one).
4. Set your allocation percentages (they should add up to 100%).
5. The tool shows exact dollar amounts for each bucket.
